# c-term

This project is my own bad version of a terminal. At the moment, I'm just rebuilding commonly used linux binaries.

This is basically one big showcase for my C suite of skills, and an attempt to learn how most (if not every) binary on linux works
under the hood.

Also, terminal emulation coming at some point.
